# Chesser-GO
Simple [PVS](https://en.wikipedia.org/wiki/Principal_variation_search) search engine written in GO


## What works
* Negamax search
* Transition tables
* Quiscence search
* Late move pruning

## Move ordering
* Hash moves
* Captures by SEE
* Promotions
* Checks
* Killer moves(2 tables)
* ...

## Evaluation
* Material
* Piece square tables
* Static exchange evaluation
* Double Pawns
* Isolated Pawns
